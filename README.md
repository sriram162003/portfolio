# Sriram M — Anime World Portfolio 🌸

A 3D anime-style runner-world portfolio. Run down a sakura-lined street — every torii gate
reveals a chapter (profile, skills, projects, certifications, contact), and the door at the
end only unlocks with a job offer 😄

**Play it:** https://sriram162003.github.io/portfolio/

Controls: ↑ run · ← → change lanes · ↓ back up · Space jump (touch d-pad + swipe on mobile).

Built with [three.js](https://threejs.org) (MIT), fully self-contained in a single `index.html`.
